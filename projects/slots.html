Slot Machine

Reel Spins and HTML5 Animation


Built with PixiJS

About

PixiJS is a rendering library that will allow you to create rich, interactive graphics, cross-platform applications, and games without having to dive into the WebGL API or deal with browser and device compatibility.

Using the pre-existing Slot Machine example available through the PixiJS docs, I spent some time reading through the code and understanding how it works before creating a variant of the game with symbols and animation logic determined by the code.
