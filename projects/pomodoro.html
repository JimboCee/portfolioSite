About

The Pomodoro Technique is a time management method develeped by Francesco Cirillo in the late 1980s. This program allows a user to set independant times for Work as well as short, frequent breaks to a structure of their choice.

Inspiration for this project came about after hearing the benefits of React and its varied usecases as a fast and reliable JavaScript Library.
